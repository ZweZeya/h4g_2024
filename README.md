# Getting Started

Install node modules
```bash
npm install
```

Set up environment variables in the ```.env``` file
```bash
cp .example.env .env
```

Run development server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
